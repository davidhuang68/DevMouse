Today I planned the main direction for my Forge project: DevMouse, a custom 3D printed productivity mouse designed for programmers.

Instead of modifying an existing mouse, I want to build the electronics myself using a custom PCB. The mouse will have normal mouse controls, plus two side macro buttons for common developer shortcuts: copy (Ctrl + C) and paste (Ctrl + V).

The goal is to make a complete wired USB mouse with a custom PCB, USB HID microcontroller, optical mouse sensor, left/right/middle click buttons, scroll wheel encoder, two side macro buttons, and a custom 3D printed ergonomic shell.

Today I focused on defining the project scope and deciding the main features. Since this project involves PCB design, firmware, CAD, and custom electronics, it fits better as a Tier 3 project instead of just a basic 3D printed mouse shell.

Next, I will research suitable parts, especially the microcontroller, optical sensor, scroll wheel encoder, and switches. After that, I will start planning the PCB layout and the physical dimensions of the shell.
